Open street map for G.N.D.E.C

This map was made for Guru Nanak dev engineering college freshers and uses Open street map libraries. By making your system an OSM tile server not only you can edit the map but can use it offline also. The core part of OSM is implemented using Mapnik library and database for rendering, mod tile and slippy for the web interface. Bash Shell Scripting has been used to automate the installation. On selecting animation a list of animations will appear, on selecting any of the places the page will move to the selected geolocation. view.animate() is used to add one or more animations. Books4You

Books4You is an android app which is aimed at for selling and buying books. It’s not a simple application for shopping books, it's a student app where students can sale out there read books to other freshers with books images attached for Users to check with the condition of the book before buying. The app is linked to the Firebase database which keeps track of the books which are uploaded for selling. The seller can sell the book by uploading a picture and other details including book name, author, description, selling price etc.. The buyer, on the other hand, can contact the selling student via the app chat function and negotiate with the seller regarding the price. Openstreet map project Documentation

This project mainly discuss about OpenStreetMap. Geographical data (geo data) is not free in many parts of the world. Generally these places have given the task of mapping to various government agencies who in return get to make money by selling the data back to you and me. The main disadvantage of Google Maps is that data is copyrighted and owned by multiple organisations like the Ordnance Survey. Google/whoever just licenses it. If we were to use it, we’d have to pay for it. This leads to the increasing demand of OSM. You can use OSM by picking an area that you know well and use the OpenStreetMap viewer to see how well the map data corresponds to your own knowledge. As on Wikipedia, it’s easy to edit, so you can help!. Also, this project is completely open source and the entire code is available to the user as and when required. There is Complete developer’s Blog reference alongwith it that helps using it a lot easier. The data and software is owned by you, the contributors. This documentation was made using Latex
